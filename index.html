SERVICE

01

New Life+ Project

より多くの人がより生きやすい社会を目指し、これをどうしても世の中に出したいという熱意を持つパートナーとプロダクトの開発に取り組みます。みんなの「こうなってて欲しい」を実現する事業です。

02

Culture+ Project

アニメ・ゲーム・音楽など、ジャンルへの愛があるメンバーが自分達の作りたいプロダクトを企画制作し提供します。
好きなことを活かしてモノを作り必要とする人から対価を得るというシンプルな事業です。

03

Creator+ Project

クリエイターの創作に役立つ情報発信やツールの開発、創作活動を支援する仕組みを作る事業です。
